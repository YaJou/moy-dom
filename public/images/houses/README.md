# Фотографии домов

У каждого объекта — **своя папка**. Имена файлов строго по номеру: `01.jpg`, `02.jpg` и т.д.

**`02.jpg`** — главное фото для карточки и галереи. **`01.jpg`** — визуализация проекта.

Формат: JPG или WebP. Рекомендуемый размер: от 1200 px по ширине.

Пока файлов нет — показывается заглушка. После загрузки фото появятся автоматически.
